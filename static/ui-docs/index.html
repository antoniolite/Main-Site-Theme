Well Icon

Class to be applied to a .fa- or .glyphicon- icon inside of a well. The icon using this class is intended to be wrapped within a small column at the -sm breakpoint and above;

   <div class="row">
     <div class="col-sm-2">
       <span class="fa fa-info well-icon"></span>
     </div>
     <div class="col-sm-10">Lorem ipsum dolor sit amet, consectetur adipisicing elit.</div>
   </div>
 </div>

Callout

Class for generic full-width callout sections. The base .callout class should NOT be used for defining background colors or inner text colors. Separate color modification classes should be used to change callout colors instead. To create a callout type with a different background/text color, create a new, separate class with as open specificity as possible (e.g. .callout-new, NOT .callout.callout-new). See .callout-dark as an example.

States

  • .callout-dark - black callout with white text.
  • .callout-gold - gold callout with white text.
  • .callout-light-gray - light gray callout with black text.
  • .callout-athletics-gold - althetics gold callout with black text.

Aside Heading

Defines a heading within a callout aside/sidebar. Designed to align with adjacent h2, h3 and h4 headings.

Example

FAQ

Lorem ipsum dolor sit amet, consectetur adipisicing elit.

      
 <div class="callout callout-light-gray">
   <aside class="callout-aside">
     <h3 class="callout-aside-heading">
       <a href="#"><span class="fa fa-question-circle"></span>FAQ</a>
     </h3>
     <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
   </aside>
 </div>

Icon Links

Links with icons

States

  • .icon-link-white - Displays a white icon and text.
  • .icon-link-white:hover - Inverse cover displayed on hover.
  • .icon-link-dark - Displays a dark icon and text.
  • .icon-link-dark:hover - Inverse cover displayed on hover.

Example

List Navigation

Basic unordered list styles with optimized list item spacing applied. Intended for use on lists of links, e.g. "Explore Further", "More Information", etc.

Example

  • List Nav 2
  • List Nav 3

   <li><a href="#">List Nav 3</a></li>
 </ul>
